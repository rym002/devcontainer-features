
# Smartthings CLI

A feature to install the smartthings CLI

## Example Usage

```json
"features": {
    "ghcr.io/rym002/devcontainer-features/smartthings:1": {
        "version": "latest"
    }
}
```

## Options

| Options Id | Description | Type | Default Value |
|-----|-----|-----|-----|
| version | CLI version or `latest` | string | latest |
